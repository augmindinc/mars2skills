### 📋 프롬프트 (최적화 버전)

심층 연구 보고서를 바탕으로 **[강화된 8단계 설득 구조]**에 따라 랜딩페이지 상세 기획안을 작성하세요.

1. **히어로 기획**: 타겟의 감성을 건드리는 강렬한 헤드라인과 시각적 컨셉 제안.
2. **문제 섹션**: 고객의 페인 포인트를 세부적으로 파헤치는 묘사.
3. **솔루션 섹션**: 핵심 기능 이점 정의.
4. **[보강] 작동 방식 (How It Works)**: 서비스 이용의 3단계 프로세스를 사용자의 관점에서 아주 쉽게 설계.
5. **[보강] 정성적 가치 (Founder's Benefit)**: 기능을 넘어 사용자가 누릴 실제 삶의 변화(시간적 여유, 심리적 안정 등)를 정성적으로 묘사.
6. **신뢰 섹션**: 후기 및 사회적 증거 배치 전략.
7. **[보강] FAQ**: 고객이 가질 수 있는 마지막 거부감(Objections) 3~5가지를 도출하고 완벽하게 답변.
8. **최종 CTA**: 전환을 완성하는 강력한 결론.

### 🚀 사용 가이드
1. 1.1단계의 조사 결과를 바탕으로 메인 헤드라인을 구성합니다.
2. **신규 보강 섹션(작동 방식, 정성적 가치, FAQ)**이 기획안의 핵심 브릿지로 기능하도록 설계합니다.
3. 전환(Conversion)을 유도하는 최적의 심리적 흐름을 구축합니다.
