### 📋 프롬프트 (입력 예정)
<!-- 여기에 직접 A/B 테스트용 프롬프트를 넣어주세요 -->
목적: 다양한 접근 방식을 테스트하기 위해 랜딩 페이지 섹션의 대체 버전을 생성합니다.

즉각적인: 
다음 랜딩 페이지 섹션에 대해 다양한 접근 방식을 테스트할 수 있도록 세 가지 버전을 제작해 주세요.

[현재 랜딩 페이지 섹션 붙여넣기]

각 변형에 대해:
다른 카피라이팅 프레임워크를 사용하세요
다른 감정적 호소를 시험해 보세요
길이와 깊이를 다르게 하세요
주요 이점 초점을 변경하세요
변형 1: 놓칠까 봐 두려워하는 심리(FOMO)와 긴급성에 초점을 맞추기
변형2: 구체적인 결과와 사회적 증거에 초점을 맞추기
변형 3: 실행 용이성과 빠른 성과에 초점을 맞추기

각 변형의 전략적 배경과 검증하고자 하는 구체적인 가설을 설명해 주십시오. 각 변형에 대한 전체 내용을 포함해 주십시오.

### 🚀 사용 가이드
1. 헤드라인, 메인 이미지, CTA 위치 등에 대해 차별화된 가설을 설정합니다.
2. 각 변형 안의 기대 효과와 가설 근거를 제시합니다.
3. 테스트 분석 시 집중해야 할 지표를 정의합니다.
